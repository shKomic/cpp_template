## how to use

### directory tree
example file tree

```
├── include
│   ├── test.hpp
│   └── module.hpp
└── src
│    ├── test.cpp
│    └── main.cpp
├── makefile
└── test
```

include: header files
src    : cpp files

### userage
commands
  `source .cppproj`    : prepare commands

  `generate {filename}` : make files (.cpp, .hpp)
  `remove   {filename}` : delete files
  `build`               : build project
  `run`                 : run   project
